GreenHarvest Agriculture is a modern agriculture web platform designed to support farmers with smart farming solutions, government subsidy information, equipment rental services, crop advisory, and an online marketplace for agricultural products.

The project includes:

User Login & Registration System
Government Schemes & Subsidy Eligibility Checker
Crop Advisory Services
Equipment Rental Services
Agricultural Product Marketplace
Shopping Cart & Checkout System
Product Selling Feature for Farmers
Responsive and Interactive UI using HTML, CSS, and JavaScript

This project aims to bridge technology and agriculture by providing farmers with easy access to farming resources, market products, and digital agricultural services in one platform.
