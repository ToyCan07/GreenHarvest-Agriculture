GreenHarvest Agriculture Website
Project Description

GreenHarvest is a modern agriculture-based website developed using HTML, CSS, and JavaScript. The main purpose of this project is to support farmers and agriculture businesses through digital technology by providing farming services, government subsidy information, crop advisory, machinery rental, and an online agricultural product marketplace.

The website includes a secure login and registration system, interactive user interface, responsive design, and various dynamic features developed using JavaScript. Users can explore farming products, rent agricultural equipment, check eligibility for government schemes, and add products to the shopping cart with checkout functionality.

This project also allows farmers to list and sell their own products directly on the platform, making it a useful digital solution for modern agriculture needs.
