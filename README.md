🌾 GreenHarvest Agriculture Website
📌 Project Overview

GreenHarvest is a modern agriculture-based web application developed using HTML, CSS, and JavaScript.
The website is designed to help farmers and users access agricultural services, government subsidy information, crop advisory, equipment rental, and an online marketplace through a simple and interactive user interface.

This mini project was developed as part of 2ES1111 – IT Workshop (ITW) at Ganpat University - U. V. Patel College of Engineering.

🎯 Purpose of the Project

The main objective of this project is to provide a digital platform for farmers where they can:

Explore farming products and machinery
Access government subsidy schemes
Get crop advisory services
Buy and sell agriculture products
Use modern agriculture solutions online

✨ Features

✔️ Login & Registration System
✔️ Responsive Single Page Website
✔️ Government Subsidy Eligibility Checker
✔️ Agriculture Product Marketplace
✔️ Add to Cart & Checkout System
✔️ Equipment Rental Services
✔️ Crop Advisory Section
✔️ Contact Form with Validation
✔️ Farmer Product Selling Feature
✔️ Interactive Animations and Effects
✔️ Dynamic Product Listing using JavaScript

🔄 Website Flow
User logs into the website
User explores services, subsidies, and products
Farmers can list their own products
Users can add products to cart
Checkout and payment process is available
Users can contact support using contact form
