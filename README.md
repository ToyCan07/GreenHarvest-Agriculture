GreenHarvest Agriculture is a modern agriculture web platform developed to support farmers with digital farming solutions and smart agricultural services. The website provides information about government subsidies, crop advisory services, equipment rental facilities, and an online marketplace for agricultural products.

This project is developed using HTML, CSS, and JavaScript with a responsive and user-friendly interface.

User Login & Registration System
Government Schemes and Subsidy Information
Subsidy Eligibility Checker
Crop Advisory Services
Equipment Rental Services
Product Marketplace
Add to Cart & Checkout System
Sell Product Feature for Farmers
Contact Form with Validation
Responsive Design for Mobile and Desktop
Interactive UI with Animations
🛠️ Technologies Used
HTML5
CSS3
JavaScript
📌 Project Purpose

The main purpose of this project is to provide farmers with a single digital platform where they can access agricultural services, buy and sell products, and get information about government farming schemes easily.
